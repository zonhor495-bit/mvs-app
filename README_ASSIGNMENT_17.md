## 🚀 Как начать

### 1. Для разработчика (локальное тестирование)

```bash
# Все установлено и готово к использованию

# Просто запустите приложение:
npm run dev
npm run electron

# Система автоматически будет проверять обновления
# Логи смотреть в %APPDATA%/MVS/startup.log
```

### 2. Для администратора (выпуск обновления)

```bash
# 1. Обновите версию в package.json
nano package.json
# Измените "version": "1.0.0" на "1.0.1"

# 2. Внесите изменения в код

# 3. Соберите приложение
npm run build

# 4. Загрузите на сервер обновлений
# Сервер: https://updates.mvs.app/releases/
# Файлы:
#   - latest.yml
#   - MVSSetup-1.0.1.exe
#   - MVSSetup-1.0.1.exe.blockmap

# 5. Готово! Пользователи получат обновление автоматически
```

### 3. Для пользователя

```
1. Откройте приложение
2. Если есть обновление → появится красивый диалог
3. Нажмите "Обновить сейчас"
4. Приложение загружает обновление в фоне (видна прогресс-бар)
5. Когда загрузится → нажмите "Перезапустить и установить"
6. Приложение обновится автоматически
✅ Готово!
```
