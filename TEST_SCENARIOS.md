## Сценарий 1: Полное восстановление всех данных

### Предусловия:
- Приложение содержит данные во всех 12 основных разделах:
  1. Клиенты (Clients)
  2. Работники (Washers)
  3. Заказы (Orders)
  4. Смены (ShiftStart)
  5. Касса (Cashier)
  6. Расходы (Expenses)
  7. Финансовые записи (Finance)
  8. Аналитика стирки (Analytics)
  9. Аналитика работников (WasherAnalytics)
  10. Тарифы (Pricing)
  11. Верность (Loyalty)
  12. Склад (Warehouse)

### Шаги теста:
1. Запустить BackupManager и создать резервную копию (кнопка "Создать резервную копию")
2. Загрузить файл backup_TIMESTAMP.zip на диск
3. Удалить все данные в приложении (очистить localStorage)
4. В BackupManager нажать кнопку "Восстановить" (⤺)
5. В мастере восстановления выбрать загруженный файл backup_TIMESTAMP.zip
6. Нажать кнопку "Проверить"
7. Убедиться, что все 12 разделов показаны в результатах валидации
8. Убедиться, что все секции отмечены для восстановления
9. Нажать "Восстановить"
10. Дождаться завершения прогресс-бара (100%)
11. После автоматической перезагрузки проверить наличие всех данных

### Ожидаемый результат:
✅ Все данные полностью восстановлены
✅ Все клиенты видны в таблице Clients
✅ Все работники видны в таблице Washers
✅ Все заказы видны в таблице Orders
✅ Все финансовые данные в Dashboard
✅ Нет ошибок валидации

### Команды валидации:
```typescript

console.log('Washers:', store.state.washers.length);
```

### Логика очистки:
```typescript
// При создании бэкапа проверяется:
if (backups.length > maxBackupCount) {
  // Удалить самый старый бэкап
  const oldest = backups.reduce((min, b) => b.createdAt < min.createdAt ? b : min);
  await deleteBackupFile(oldest);
}
```

## Сценарий 10: Облачное хранилище (архитектура)

### Статус: Архитектура готова (реализация в будущих версиях)

### Поддерживаемые провайдеры:
1. Google Drive (через Google Drive API v3)
2. OneDrive (через Microsoft Graph API)
3. Dropbox (через Dropbox API)
4. Custom Server (через собственное API)

### Интерфейсы и типы добавлены в `src/types.ts`:

```typescript
type CloudProvider = 'google-drive' | 'onedrive' | 'dropbox' | 'custom-server';

interface CloudSyncConfig {
  provider: CloudProvider;
  enabled: boolean;
  authToken: string;
  folder?: string;
  autoSync: boolean;
  syncInterval?: number; // в миллисекундах
  uploadCompressionEnabled: boolean;
}

interface CloudSyncStatus {
  provider: CloudProvider;
  isConnected: boolean;
  lastSyncAt: string; // ISO timestamp
  pendingSyncs: number;
  storageLimitBytes: number;
  usedStorageBytes: number;
}

interface CloudFile {
  id: string;
  name: string;
  size: number;
  modifiedAt: string;
  provider: CloudProvider;
  isEncrypted: boolean;
  remoteId: string;
}

interface CloudSyncLog {
  timestamp: string;
  provider: CloudProvider;
  operation: 'upload' | 'download' | 'delete' | 'list';
  status: 'success' | 'failed' | 'pending';
  bytesTransferred?: number;
  errorMessage?: string;
}
```

```typescript
// Поддерживаемые провайдеры:

- CustomServerProvider
```
