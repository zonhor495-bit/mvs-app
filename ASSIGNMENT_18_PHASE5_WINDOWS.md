# Assignment 18 Phase 5: Тестирование на Windows 10/11

## 🖥️ Цель Phase 5

Подтвердить что система обновлений работает корректно на разных версиях Windows:
- Windows 10 (все актуальные версии)
- Windows 11
- Сохранение ярлыков
- Сохранение настроек
- Сохранение базы данных
- Корректный запуск после обновления

---

## 📋 Требования для тестирования

### Оборудование

- [ ] ПК с Windows 10 (версия 1909 или новее)
- [ ] ПК с Windows 11 (любая версия)
- [ ] Оба ПК подключены к интернету
- [ ] На обоих установлена версия 1.0.0

### Программное обеспечение

- [ ] Node.js 16+ (для разработки)
- [ ] npm (включена в Node.js)
- [ ] Visual Studio Build Tools (для компиляции)
- [ ] Git (для управления версиями)

## 🔍 Специфические проверки Windows 10/11

### Проверка реестра (Registry)

```powershell
# Проверить что приложение зарегистрировано в Programs
# Windows 10/11 → Settings → Apps → Apps & features
# Должно быть: "MVS — Car Management System"

# Проверить uninstall entry
reg query "HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall" | findstr MVS
```

### Проверка файловой системы

```powershell
# Проверить что файлы установлены правильно
# Обычно в: C:\Program Files\MVS\

dir "C:\Program Files\MVS"

# Должны быть:
# - MVS.exe (или MVS-win32-x64\...)
# - Ресурсы приложения
# - Конфиг файлы
```

### Проверка AppData

```powershell
# Проверить что данные сохраняются правильно
# Обычно в: %APPDATA%\MVS\

dir "%APPDATA%\MVS"

# Должны быть:
# - startup.log (логи приложения)
# - Database (база данных)
# - Другие конфиг файлы
```
