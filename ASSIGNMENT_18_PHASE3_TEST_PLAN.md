## 🔍 Логи для проверки

### Где найти startup.log

**Windows:**
```
%APPDATA%/MVS/startup.log
```

**macOS:**
```
~/Library/Application Support/MVS/startup.log
```

**Linux:**
```
~/.config/MVS/startup.log
```

### Что искать в логах

```
✓ [updater] Initialized auto-updater
✓ [updater] Check for updates...
```
